# Integral Calculator
Приложение для вычисления определенных интегралов


## Требования 
- node >= 12.22.9
- npm >= 8.5.1
- vite >= 3.1.4

## Начало Работы
``` 
$ npm install  
```
Локальный сервер для разработки:
``` 
$ npm run dev 
```
Проверки ESlint:
``` 
$ npm run lint 
```
Запуск тестов:
``` 
$ npm run test 
```
